# Carbon-Footprint-Tracker
An app that tracks  and calculates a user's carbon footprints based on its daily activities and provides personalized suggestions.

#Features
-Allows users to input daily activities (like travel, electricity, food habits).
-Calculates estimated CO‚ÇÇ emissions based on provided data.
-Provides personalized tips and suggestions to reduce carbon footprint.
-Displays results in a clean, user-friendly interface.
-Stores data for users to track progress over time.

#Technologies Used
-Frontend: HTML5 ,CSS3 ,JavaScript, Chart.js
-Backend: Java, Servlets, JSP, MySQL
-Server: Apache Tomcat 9.0
-IDE Used: Eclipse


# How To Run
1.Clone the repository to your system:
git clone https://github.com/Sakshi28738/Carbon-Footprint-Tracker.git

2.Open the project in Eclipse.
3.Configure Apache Tomcat server.
4.Set up the MySQL database using your DB credentials.
5.Run the application on the server.

6.Open your browser and go to:
http://localhost:8080/CarbonFootprintTracker
